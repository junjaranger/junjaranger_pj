### 코드실행방법

#### Front-end

##### 폴더구조

/src/main/frontend => root
@/pages/index => localhost:3000/
@/components => 컴포넌트 폴더
@/pages/fonts => 로컬 폰트 폴더
@/pages/api => api 모음 폴더
@/public => 이미지, 아이콘, 파일 등의 폴더

##### 명령어

```bash
/* 프론트 서버 루트로 이동 */
cd ./src/main/frontend 

/* 프론트엔드 서버 실행 */
pnpm run dev 

/* 오타 및 에러 잡기 명령어 */
pnpm run lint --fix 
```
